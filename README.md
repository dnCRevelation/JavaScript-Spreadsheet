# Functional JS Spreadsheet that works similarly to Excel but minimalistic

### Command Syntax In Cells: "=FUNCTION(Cell:Cell)"
### Example: "=SUM(A1:B1)"

Command List:
SUM      LASTTWO
AVERAGE  HAS2
MEDIAN   INCREMENT 
EVEN     RANDOM 
SOMEEVEN  RANGE
EVERYEVEN  NODUPES
FIRSTTWO
